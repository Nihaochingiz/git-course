#  Reset – Мягкий reset --soft: замена и объединение коммитов

- Мягкий reset

``` 
git reset --soft @~
git branch -v
```
Указатель на мастер перешел на один коммит назад

- Вернуться на один коммит обратно

```
git reset --soft ORIG_HEAD
``` 

- Вернуть master на предыдущий коммит

```
git reset --soft @~
```
